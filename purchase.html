  • PCを買い替えた場合は？What if I change my PC?
    新しいPCのハードウェアIDをお知らせいただければ、無料で再発行いたします。 Contact us with your new Hardware ID for a free re-issue.
  • Microsoft Store版との違いは？What's the difference from the Microsoft Store version?
    機能は同じです。Store版はMicrosoft Store経由の自動インストール・更新、ライセンス管理不要。ダウンロード版はセットアップファイルで手動インストール＋ライセンスキー入力です。 Features are identical. Store version auto-installs via Microsoft Store with no license management. Download version requires manual installation + license key entry.

    時間制限はありません。動画長5分以内であれば、何度でも無料でお使いいただけます。 No time limit. You can use it free indefinitely for videos under 5 minutes.
