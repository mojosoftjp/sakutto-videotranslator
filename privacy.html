さくっと動画翻訳Snap Video Translator

はじめに

「さくっと動画翻訳」（以下「本アプリ」）は、ユーザーのプライバシーを尊重して設計されています。本ポリシーでは、本アプリにおける情報の取り扱いについて説明します。

個人情報の収集について

本アプリは、ユーザーの個人情報を収集しません。

  • 氏名、メールアドレス、電話番号などの個人情報は収集しません
  • 利用状況やアクセスログなどの分析データは収集しません
  • Cookie やトラッキング技術は使用しません
  • 開発元（Mojosoft）がユーザーのデータを受け取ることはありません

文字起こし（ローカル処理）

動画・音声の文字起こしは、OpenAI Whisper をユーザーのPC上で実行します。音声データが外部サーバーに送信されることはありません。

AI翻訳について

AI翻訳機能を有効にした場合、字幕テキストは、ユーザーが選択したAIサービスに送信されます。

  • クラウドのAIサービス（Gemini・OpenAI・Claude など）を選択した場合、字幕テキストは当該サービスに送信されます。送信内容は各サービスのプライバシーポリシーに従って扱われます。
  • OpenAI互換のローカルLLMを選択した場合、データはユーザーの環境内で処理されます。
  • AIサービスのAPIキーは、ユーザーのPC内（Windowsレジストリ）にのみ保存され、選択したサービスへの接続にのみ使用されます。開発元に送信されることはありません。

AI吹き替えについて

AI吹き替え機能を使用した場合、翻訳後のテキストは、音声を生成するために Microsoft のオンライン音声合成サービス（Edge TTS）に送信されます。

ネットワーク通信について

文字起こしはローカルで完結します。AI翻訳・AI吹き替え・文字起こしモデルのダウンロードを行う際にはインターネット接続が必要です。本アプリがネットワーク通信を行うのは、これらの機能をユーザーが利用するときに限られます。

使用しているオープンソースソフトウェア

本アプリは動画処理エンジンとして FFmpeg（GPL版）を使用しています。FFmpegのソースコードは https://ffmpeg.org/ で入手できます。

第三者への提供

開発元はユーザーのデータを収集しないため、開発元から第三者へのデータ提供はありません。AI翻訳・AI吹き替えでユーザーが利用する外部サービスについては、各サービスのプライバシーポリシーをご確認ください。

お子様のプライバシー

本アプリは個人情報を収集しません。

本ポリシーの変更

本ポリシーは、必要に応じて更新される場合があります。変更があった場合は、本ページにて更新日とともに公開します。

Introduction

"Snap Video Translator" (hereinafter "the App") is designed with respect for user privacy. This policy explains how information is handled within the App.

Collection of Personal Information

This App does not collect personal information.

  • No personal data such as names, email addresses, or phone numbers is collected
  • No usage analytics or access logs are collected
  • No cookies or tracking technologies are used
  • The developer (Mojosoft) does not receive any user data

Transcription (Local Processing)

Transcription of video and audio is performed by OpenAI Whisper on the user's PC. Audio data is never sent to an external server.

AI Translation

When the AI translation feature is enabled, subtitle text is sent to the AI service chosen by the user.

  • If a cloud AI service (such as Gemini, OpenAI, or Claude) is selected, the subtitle text is sent to that service and handled under that service's own privacy policy.
  • If an OpenAI-compatible local LLM is selected, the data is processed within the user's own environment.
  • The AI service's API key is stored only on the user's PC (Windows registry) and used solely to connect to the selected service. It is never sent to the developer.

AI Dubbing

When the AI dubbing feature is used, the translated text is sent to Microsoft's online text-to-speech service (Edge TTS) in order to generate the voice audio.

Network Communication

Transcription is completed locally. An internet connection is required for AI translation, AI dubbing, and downloading transcription models. The App performs network communication only when the user uses these features.

Open Source Software Used

This App uses FFmpeg (GPL version) as its video processing engine. FFmpeg source code is available at https://ffmpeg.org/.

Third-Party Sharing

Since the developer does not collect user data, there is no data sharing from the developer to third parties. For the external services used in AI translation and AI dubbing, please review each service's own privacy policy.

Children's Privacy

This App does not collect personal information.

Changes to This Policy

This policy may be updated as needed. Any changes will be published on this page with the updated date.
